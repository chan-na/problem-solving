## 스택을 사용해서 직접 되는지 안되는지 해보는 방법

- 스택을 사용하는 간단한 문제라고 생각했지만, 구현하는데 있어서 생각보다 상태가 여러개라서 복잡했다.
  - 현재 스택에서 나와야할 coach 번호, 스택의 top, 스택으로 곧 push될 coach번호를 관리해야한다.
